# cloud-kitchen-git-playground
Practice repo for Git &amp; GitHub internship batch

Update repository remote URL : git remote set-url  origin https://github.com/saikiran290/cloud-kitchen-git-playground.git

Create a working branch : git checkout -b "branchsk"

Add changes : git add .

Commit updates  : git commit -m "aws notes added"

Push branch to GitHub  : git push origin branchsk
